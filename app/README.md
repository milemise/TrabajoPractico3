TRABAJO PRACTICO 3

Este proyecto consiste en una aplicación desarrollada en Android que permite la búsqueda y visualización de libros "El hobbit, Crimen y Castigo, Orgullo y Prejuicio y 1984".

Tecnologías y herramientas aplicadas:
Arquitectura MVVM: Implementé un ViewModel con LiveData para manejar la lógica de búsqueda y la lista de libros de forma separada a la vista.

Glide: Integré esta librería externa para la gestión y carga de imágenes mediante URLs.

Material Design: Usé componentes como ChipGroup para los géneros y un diseño basado en ScrollView para que la info sea fácil de leer.

View Binding: Para una vinculación de vistas más segura y limpia.

Intents & Serializable: Paso de objetos completos entre actividades.
